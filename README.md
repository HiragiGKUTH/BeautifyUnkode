# BeautifyUnkode

web application for my class

## What's this

## Function One

This application beautify your "Unkode" (means shit code)

example:

```cpp:shit.cpp
#include <stdio.h>

int main() {
printf("hoge");
return 0;
}
```

to

```cpp:beauty.cpp
#include <stdio.h>

int main() {
    printf("hoge");
    return 0;
}
```


## Feature Two

example:
